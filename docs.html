📚 Statsnap Documentation

Professional creative editor for social media content, photo editing, and collage creation.

🎯 Modes

✏️ Design Mode

Create graphics with text, images, shapes, and stickers. Full layer management with drag, resize, rotate, and z-index control.

🖼 Photo Mode

Edit photos with professional tools: brightness, contrast, saturation, hue, filters, paint brush, blur brush, and crop.

🎭 Collage Mode

Create photo collages with 2, 3, or 4 grid templates. Canvas is fixed; images are placed in cells. Supports free-form collage.

🛠 Tools Reference

Design Mode Tools

Text - Add editable text layers
Image - Import images from device
Shape - Add colored shape layers
Sticker - Add emoji stickers
Background - Change canvas background
Layers - View and manage layers

Photo Mode Tools

Add Photo - Import photo for editing
Adjust - Brightness, contrast, saturation, hue
Filters - 8 preset filters (Grayscale, Sepia, Vintage, etc.)
Crop - Use resize handles to crop
Paint - Draw on images with color brush
Blur - Blur specific areas of images

Collage Mode Tools

2 Grid - Two-cell grid layout
3 Grid - Three-cell grid layout
4 Grid - Four-cell grid layout
Free - Free-form collage
Background - Change canvas background

Layer Tools (when layer selected)

Lock - Prevent layer from moving
Duplicate - Create copy of layer
Opacity - Adjust layer transparency
Blend - Change blend mode (multiply, screen, overlay, etc.)
Front - Bring layer to front
Back - Send layer to back

⌨️ Keyboard Shortcuts

Ctrl+ZUndo Ctrl+Shift+ZRedo Ctrl+PlusZoom in Ctrl+MinusZoom out Ctrl+0Reset zoom DeleteDelete selected layer EscapeDeselect / Stop brush ScrollZoom in/out

💡 Tips

  • Switch modes anytime - your work is preserved
  • Lock layers to prevent accidental movement
  • Use pinch gesture on touch devices to zoom
  • Canvas size can be changed from the hamburger menu
  • Projects auto-save every 30 seconds
  • Export as JPEG or PNG from the menu
